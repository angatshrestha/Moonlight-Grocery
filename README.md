# Moonlight Grocery – Online eCommerce Platform 🌙🛒

A modern eCommerce web application designed to provide a seamless online grocery shopping experience. Moonlight Grocery focuses on user-friendly design, real-time product browsing, and efficient order management.

---

## Project Overview
Moonlight Grocery is an online platform where customers can browse grocery products, add items to their cart, and place orders. The system is designed to be responsive, fast, and visually appealing with a premium dark (moonlight) theme.

This project aims to simplify grocery shopping by providing:
- Easy product discovery  
- Smooth checkout experience  
- Real-time order tracking (future scope)  
- Customer and supplier management  

---

## Features
- 🛍️ Browse products by category  
- 🔍 Search functionality  
- 🛒 Add to cart & manage cart  
- 💳 Secure checkout system (planned)  
- 👤 User authentication (login/signup)  
- 📦 Order management system  
- 🎨 Dynamic UI with dark purple theme  
- 📱 Fully responsive design
